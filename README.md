# No Augment Negative Effects
A mod for Rogue Trader which does the following:
- Removes all the negative effects of Augments in Rogue Tader. 
- Removes the "Positive Effects" header in the description of Augments.
## How to Install:
Place the zip file contents in C:\Users\\(Username)\AppData\LocalLow\Owlcat Games\Warhammer 40000 Rogue Trader\UnityModManager\
