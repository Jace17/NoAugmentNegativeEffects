# No Augment Negative Effects
A mod for Rogue Trader which does the following:
- Removes all the negative effects of Augments. 
- Removes the "Positive Effects" header in the description of Augments.
- Augments that have been installed on a character before the mod have to re-equipped to remove their negative effects.
## How to Install:
- Use ModFinder: https://github.com/CasDragon/ModFinder to automatically install **OR**
- Manually place the zip file contents in C:\Users\\(Username)\AppData\LocalLow\Owlcat Games\Warhammer 40000 Rogue Trader\UnityModManager\
